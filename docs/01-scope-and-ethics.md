# Scope and Ethics

## Purpose

This project was created for educational purposes as part of a cybersecurity learning challenge.

The goal is to study brute-force risks, authentication weaknesses, and defensive measures in a controlled lab environment.

## Authorized Scope

All tests are planned to be performed only in a local lab environment using virtual machines.

The environment includes:

- Kali Linux;
- Metasploitable 2;
- DVWA;
- VirtualBox;
- Host-only network.

## Out of Scope

The following activities are not part of this project:

- Testing public websites;
- Testing third-party networks;
- Testing corporate systems;
- Using real credentials;
- Collecting personal data;
- Publishing real passwords;
- Attacking unauthorized systems.

## Ethical Commitment

This project is focused on learning, documentation, and prevention.

The objective is not to cause damage or gain unauthorized access, but to understand security risks and propose mitigation strategies.
